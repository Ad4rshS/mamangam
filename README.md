# Mamangam

A 100% free-to-play fantasy sports app for IPL 2026. Create private leagues, build your dream team with your favorite players, and compete with friends with real-time live score updates.

## Features
- Real-time IPL 2026 match data
- Official Playing XI lineups with AI grounding
- Team creation and editing (no credit limits)
- Private contests and leaderboards
- Animated UI with Framer Motion
